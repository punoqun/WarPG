## Element System

### Elements
1. **Physical** - Standard melee attacks
2. **Fire** üî• - Effective against ice/nature
3. **Ice** ‚ùÑÔ∏è - Effective against fire creatures
4. **Lightning** ‚ö° - Effective against flyers and constructs
5. **Poison** ‚ò†Ô∏è - Damage over time
6. **Holy** ‚ú® - Effective against undead and demons
7. **Dark** üåë - Effective against holy creatures
8. **Nature** üåø - Effective in natural environments

### Damage Modifiers
- **Weakness:** 150% damage (50% bonus)
- **Normal:** 100% damage
- **Resistance:** 50% damage (50% reduction)

## Strategy Tips

### 1. Know Your Elements
- Check monster weakness on encounter
- Carry multiple weapon types when possible
- Avoid zones where your attacks are resisted

### 2. Zone Matching
- Match zone to your level
- Don't skip ahead too fast
- Grinding in easier zones is valid

### 3. Equipment Planning
- Fire weapons for Tundra/Swamp/Forest
- Ice weapons for Volcano/Desert
- Holy weapons for Ruins/Dungeon
- Lightning weapons for Mountain

### 4. Risk vs Reward
- Higher zones = better loot
- But also tougher enemies
- Balance risk with current power level

### 5. Zone Rotation
- Zones cycle as you progress
- Early zones: More variety
- Mid zones: Mixed difficulty
- Late zones: Dungeons more common

## Technical Implementation

### Database Schema
```sql
CREATE TABLE zones (
    id BIGINT PRIMARY KEY,
    zoneNumber INT NOT NULL,
    zoneType VARCHAR(255) NOT NULL,
    minLevel INT NOT NULL,
    maxLevel INT NOT NULL,
    customName VARCHAR(100)
);

CREATE TABLE monsters (
    -- existing fields --
    preferredZone VARCHAR(255),
    weakness VARCHAR(255),
    resistance VARCHAR(255),
    specialDropRate INT
);
```

### Zone Determination Algorithm
Zones are generated dynamically based on progressive difficulty:
1. Zones 1-2: Grassland (tutorial)
2. Zones 3-4: Grassland/Forest mix
3. Zones 5-6: Desert/Swamp (element introduction)
4. Zones 7-8: Mountain/Cave (harder combat)
5. Zones 9-10: Tundra/Ruins (advanced)
6. Zones 11-15: Mixed mid-game variety
7. Zones 16+: Dungeon-heavy end-game

### Monster Spawning
```java
Monster monster = monsterDAO.getRandomMonsterForZone(
    currentZone.getZoneType(), 
    currentPlayer.getLevel()
);
```

Monsters spawn based on:
1. Zone type preference
2. Player level ¬±1
3. Random selection from matching pool
